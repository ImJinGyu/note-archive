# Changelog

## [0.1.0] - 2026-03-17

### 추가
- 노트 CRUD (생성, 조회, 소프트 삭제)
- 휴지통 기능 (복구, 영구 삭제, 휴지통 비우기)
- 노트 잠금 (bcrypt 비밀번호 암호화)
- 탭 구성 (추가, 삭제, 이름 변경)
- 10가지 블록 유형
  - 텍스트 (Markdown)
  - 코드 (구문 강조)
  - 팁 박스
  - 단계별 가이드
  - 테이블
  - 체크리스트
  - 파일 첨부 (미리보기, 다운로드)
  - 키워드 조건 블록
  - 흐름도 블록
  - 기능 목록 블록
- 블록 선택 모달 호버 미리보기
- 블록 선택 페이징 (7개 초과 시)
- 편집 → 완료 버튼 일괄 저장 패턴
- 글라스모피즘 UI (하늘색 테마)
- Toast 알림
- 태그 / 설명 / 아이콘 지원
