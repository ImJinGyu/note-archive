## 메인 페이지
- 등록된 노트들을 카드 형태로 보여줘
- 각 카드에 표시되는 정보:
  - 아이콘
  - 타이틀
  - 설명
  - 태그
  - 암호 설정된 노트는 자물쇠 아이콘 표시
- 새 노트 추가 버튼
- 카드 클릭하면 노트 상세 페이지로 이동
- 암호 설정된 노트 클릭 시 비밀번호 입력 모달 표시
- 일반 노트 / 암호 노트 분류해서 표시 (탭 또는 섹션으로 구분, 디자인은 알아서)

---

## 노트 등록 페이지
- 아이콘 선택 (이모지 팔레트에서 선택하는 방식, 직접 업로드 아님)
- 타이틀 입력
- 설명 입력
- 태그 입력 (여러 개 가능)
- 암호 설정 체크박스 (체크해야 비밀번호 입력창 활성화, 미체크 시 입력창 비활성화)
- 저장 버튼

---

## 노트 상세 페이지
- 상단에 탭으로 섹션 구분
- 탭 추가 / 삭제 / 이름 변경 가능
- 각 탭 안에 아래 블록들을 자유롭게 추가 가능
- 블록 순서 변경 가능

### 블록 공통
- 모든 블록에 타이틀 사용 / 미사용 토글 기능 (기본값은 미사용, 토글 켜면 타이틀 입력창 활성화)

### 블록 종류 7가지
1. **텍스트 / 마크다운** - 일반 텍스트 또는 마크다운 작성
2. **코드 블록** - 코드 표시 + 복사 버튼 포함
3. **팁 박스** - 아이콘 + 불릿 리스트 형태
4. **단계별 설명** - 번호 + 설명 + 코드 형태
5. **표** - 컬럼명 자유롭게 설정 가능
6. **체크리스트** - 항목별 체크 + 전체 진행률 표시
7. **파일 첨부** - 파일 업로드 가능, 상단에 파일명 표시, 파일 내용을 인라인으로 미리보기 가능 (열기/닫기 토글), 다운로드 버튼 포함

### 블록 미리보기
- 각 블록은 편집 모드 / 미리보기 모드 전환 가능
- 미리보기 버튼 클릭하면 실제 화면에서 어떻게 보이는지 바로 확인 가능
- 전체 탭 미리보기도 가능 (모든 블록을 한 번에 미리보기)
